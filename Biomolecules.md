# Biomolecules

## TL;DR:


## Definition:
While there are 92 elements in the periodic table, most biomolecules only contain six different ones:
- [[Oxygen]]
- [[Carbon]]
- [[Hydrogen]]
- [[Nitrogen]]
- [[Phosphorous]]
- [[Sulfur]]
[^1]

*"These six elements form a limited number of [[functional groups]]"*[^1]
